⚙️ SharePoint Settings

Update your SharePoint site URL. The list name must be TeamCallData.

📋 SharePoint List setup checklist:
1. Go to your SharePoint site → Site Contents → New → List
2. Name it exactly: TeamCallData
3. Add these columns (all Single line of text type):
  AgentName, Status, StatusDuration, CallNumber, CallDuration, CallChannel, EntryDate, AgentSID, Provider, Department, AddedBy
4. Share the list with all colleagues (at minimum: Contribute permission)
5. Admins who can delete: give them Edit or Full Control
